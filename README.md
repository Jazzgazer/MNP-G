## 📱 프로젝트 소개
밋치리네코팝 시즌2 클라이언트, 운영툴 소스 코드입니다. 


- MNP-G : 자체 서비스 
- MNP-QROAD : 동남아 서비스(QROAD 퍼블리싱)
- OperationTool : 운영툴

### ⚙️ 개발 환경
- Unity Engine 5.6.1
- AWS EC2(Node.js)
- AWS RDS MySQL
- Nginx
- AWS S3
